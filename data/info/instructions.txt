- Για την εκτέλεση του προγραμματος με γραφική διεπαφή απαιτείται το module Pillow https://pypi.org/project/Pillow/
  Για την εγκατάσταση του πληκτρολογήστε την εντολή: 'pip install Pillow'

- Για να τρέξετε την εφαρμογή μέσω γραμμής εντολών πληκτρολογήστε στην γραμμή εντολών: Εργαστηριακή-Άσκηση-1\src> python main.py cmd

- Για να τρέξετε την εφαρμογή μέσω γραφικής διεπαφής χρήστη πληκτρολογήστε στην γραμμή εντολών: Εργαστηριακή-Άσκηση-1\src> python main.py gui

- Για unit-testing πληκτρολογήστε στην γραμμή εντολών: Εργαστηριακή-Άσκηση-1\src> python unit_testing.py
